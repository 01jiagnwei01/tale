# Tale

做有逼格的Java博客程序。

## 如何使用

1. 建立你的博客数据库
2. 下载最新版 [Tale]() 发布包
3. 执行 `java -jar tale-xxx.jar`
4. 访问 http://host:port/install 进行安装
